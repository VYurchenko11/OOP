Игра в жизнь:
Существует три режима игры:
    1. offline - в этом режиме загружается определенный файл, указывается кол-во итераций, а также выходной файл, программа записывает новую вселенную в этот файл.(можно вывести с помощью input file)
        Запуск:
            .\GameOfLife <input_file> -i <iterations> -o <output_file>
    2. default - в этом режиме автоматически запускается файл input.life, выводится на консоль, а затем передается управление пользователю
        Запуск:
            .\GameOfLife
    3. input file - в этом режиме загружается определенный файл, выводится в консоль, затем управление передается пользователю
        Запуск:
            .\GameOfLife <input_file>
